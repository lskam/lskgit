一、光盘内容

    本光盘是《Linux环境下C编程指南（第2版）》一书的配书光盘，内容包括了书中所有示例的源程序代码。 

    本光盘根目录下包括文件夹source code和一个readme文本文件，其内容和含义说明如下:

    (1) source code目录中包含了本书所有章节示例的源代码。各个章节内容分别置于以chapter_x命名的文件夹，各个文件夹包含对应章节出现的示例代码，示例代码建议用写字板打开。

    (2) readme文本文件是本书光盘的使用说明。 


二、运行环境
    1. 硬件要求
     CPU：1.6G以上，内存：1G以上，硬盘：32G以上。 
 
    2. 安装Linux操作系统及开发包
     本书的示例程序，已经在基于Linux的Fedora 10平台上成功的测试。
     Fedora 10 dvd版安装: 按照提示步骤，磁盘分区结束后，会提示选择是否安装附带的开发包以及工具，选择“现在定制”，然后在列出
的项目中选中“选择安装automake 工具和gcc/g++编译器”。然后开始安装，安装结束后即可运行本书中的程序，无需其他配置。
     Fedora 10自带的gcc版本为4.3.2，make版本为3.81。
